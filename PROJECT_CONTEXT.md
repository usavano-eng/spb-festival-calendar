# Контекст проекта — Festival Calendar SPB

## Что это
Сайт-календарь гик/аниме/ролевых фестивалей Санкт-Петербурга.
Деплоится на GitHub Pages через GitHub Actions.

## Архитектура
- `build_calendar.py` — парсер VK API + генератор HTML/CSV/JSON
- `update-calendar.yml` — GitHub Actions workflow (запуск по понедельникам + ручной)
- `public/` — артефакты для деплоя (создаётся скриптом)

## Ключевые решения
- HTML-шаблон: `string.Template` (не `.format()`!) — избегает конфликта с CSS-переменными `--bg`
- VK-парсер: `wall.get` → фильтр по дате публикации поста + явный год в тексте
- Fallback: `SAMPLE_FESTIVALS` если VK_TOKEN не задан или API недоступен
- Дата в шапке файла: `build_calendar.py` содержит `# Generated: ...` для отслеживания версий

## Секреты
- `VK_TOKEN` — токен VK API, хранится в GitHub Secrets
